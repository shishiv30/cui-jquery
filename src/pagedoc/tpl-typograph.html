The increase/decrease rate is 0.618
level1: f1 = f3 / 0.618 = f3 + f3 * 0.618
level2: f2 = (f1 - f3) * 0.382 + f3
level3: f3 = f5 / 0.618 = f5 + f5 * 0.618
level4: f4 = f5 + (f3-f5) * 0.382
level5: f5 = $font-size-base
level6: f6 = f7 + $g0 / 2
level7: f7 = 1 * 0.618
