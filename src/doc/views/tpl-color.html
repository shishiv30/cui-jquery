color solution the regular color we used.

The color solution can be group by emotion and context.

Main color is the regular hightlighted color for the whole system.

[Use case] link color, regular button background, header background,footer background and some hightlighted element.

Mid color is applied in the context which about minorcase.

Regular gray color: The border color; The text color; Lightgray color; the disable color;

Tip color is applied in the context which about curious.

[Use case] tip info, some special case which is outstanding but not improtant.

safe color is applied in the context which about fulfillment, confident and benefit.

[Use case] success info, high rank and any infomation which is good to know.

danger color is applied in the context which about seriousness.

[Use case] error info, low rank, dangerous behavior and something unexpected.

warning color is applied in the context which about peeking.

[Use case] low level element or nice to have content.
